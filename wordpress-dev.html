Simple Brochure Website and Droplet Setup

An outside agency contact me to complete this simple project.

It involved setting up a droplet with Digital Ocean (LAMP) and developing a WordPress site

Project Features

As per client's need I setup the droplet, installed WordPress with a purchased theme and converted their design from PSD to WordPress.

Front End Features

  • Converted PSD files and Implemented in WordPress
  • Implemented some cusom JS features not available in the 'off the shelf' theme.
  • A few custom and seasonal forms.

Other Features

  • Droplet setup with LAMP stack, Let's Encrypt, WordPress etc.
  • Integrated with offsite MySQL database.
  • All forms integrated with clients Zoho CRM.
